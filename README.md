Приложение для работы с сущностями организации: работники, проекты, должности. Осуществляет CRUD операции над ними, а также:
1. Поиск всех работников по названию должности.
2. Поиск всех проектов по имени и фамилии работника.
3. Выводит список сотрудников, задействованных на каждом проекте.

Используемые технологии: Spring Core, Spring MVC, Hibernate, Servlet, Maven, PostgreSQL, Lombok. Схема базы данных приведена в файле ERD_OrgAston_db.jpg
